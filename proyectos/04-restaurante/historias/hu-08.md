# HU-08: No liberar mesa con comanda abierta

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** sistema  
**Quiero** impedir liberar una mesa con comanda abierta sin cerrar  
**Para** no perder pedidos pendientes

## Criterios de aceptación

- [ ] Dado comanda abierta con ítems, cuando intento liberar mesa sin cerrar, entonces la operación falla o exige cerrar primero.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Mesa con comanda abierta.
2. Intentar liberar sin cerrar.
3. Verificar bloqueo o flujo que exige cierre.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
