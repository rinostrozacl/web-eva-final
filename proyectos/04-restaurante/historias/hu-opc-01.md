# HU-OPC-01: Vista cocina

## Obligatoriedad

Opcional (bonus)

## Historia de usuario

**Como** cocina  
**Quiero** ver ítems pendientes y marcarlos como en preparación o listo  
**Para** coordinar con el salón

## Criterios de aceptación

- [ ] Dado ítems pedidos, cuando abro vista cocina, entonces veo pendientes.
- [ ] Dado ítem, cuando cambio estado a listo, entonces se refleja en cocina y/o mozo.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Pedir ítems en comanda.
2. Abrir vista cocina.
3. Cambiar estado de ítem y verificar.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
5. Bonus: **+5 pts** si cumple; **0** si no implementado (sin descuento).
