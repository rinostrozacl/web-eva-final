# HU-05: Agregar ítems a comanda

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** mozo  
**Quiero** agregar ítems del menú a la comanda  
**Para** registrar lo pedido

## Criterios de aceptación

- [ ] Dado comanda abierta, cuando agrego ítems, entonces aparecen con cantidad y subtotal.
- [ ] Dado varios ítems, entonces el subtotal parcial se actualiza.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Abrir comanda.
2. Agregar 2 ítems del menú.
3. Verificar líneas y subtotales.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
