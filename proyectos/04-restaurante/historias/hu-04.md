# HU-04: Abrir comanda

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** mozo  
**Quiero** abrir una comanda en una mesa libre  
**Para** tomar el pedido

## Criterios de aceptación

- [ ] Dado mesa libre, cuando abro comanda, entonces la mesa pasa a ocupada y existe comanda abierta.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Seleccionar mesa libre.
2. Abrir comanda.
3. Verificar mesa ocupada y comanda activa.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
