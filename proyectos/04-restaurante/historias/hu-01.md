# HU-01: Registrar mesas

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** administrador  
**Quiero** registrar mesas con número, capacidad y estado  
**Para** organizar el salón

## Criterios de aceptación

- [ ] Dado modelo Mesa, entonces incluye número, capacidad y estado (libre/ocupada).

## Requiere interfaz web

Parcial

## Procedimiento de verificación (corrector)

1. Revisar modelo Mesa o crear vía web.
2. Verificar campos y persistencia.
3. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
