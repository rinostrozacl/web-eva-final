# HU-01: Registrar productos

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** administrador  
**Quiero** registrar productos con nombre, precio y stock  
**Para** controlar el inventario

## Criterios de aceptación

- [ ] Dado modelo Producto, cuando reviso campos, entonces incluye nombre, precio y stock.
- [ ] Dado producto creado, entonces persiste en BD.

## Requiere interfaz web

Parcial

## Procedimiento de verificación (corrector)

1. Revisar modelo o crear producto vía API/web.
2. Verificar campos y persistencia.
3. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
