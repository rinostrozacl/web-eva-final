# HU-02: Gestionar productos desde la web

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** administrador  
**Quiero** crear, editar y eliminar productos desde la web  
**Para** mantener el catálogo

## Criterios de aceptación

- [ ] Dado admin autenticado, cuando CRUD producto en web, entonces cambios persisten en listado.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Login admin.
2. Crear, editar y eliminar producto de prueba.
3. Verificar listado actualizado.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
