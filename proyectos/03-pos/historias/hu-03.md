# HU-03: Pantalla POS con catálogo

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** cajero  
**Quiero** una pantalla POS con el catálogo de productos visible  
**Para** vender rápidamente

## Criterios de aceptación

- [ ] Dado productos en catálogo, cuando abro POS, entonces veo productos con nombre y precio.
- [ ] Dado la pantalla POS, cuando navego, entonces puedo seleccionar productos.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Abrir pantalla POS/mostrador.
2. Verificar catálogo visible con precios.
3. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
