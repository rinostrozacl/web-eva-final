# HU-04: Carrito de venta

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** cajero  
**Quiero** agregar productos a un carrito de venta  
**Para** armar el pedido del cliente

## Criterios de aceptación

- [ ] Dado productos en catálogo, cuando agrego al carrito, entonces aparecen con cantidad y subtotal.
- [ ] Dado ítems en carrito, cuando modifico cantidad, entonces el subtotal se actualiza.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Agregar 2 productos al carrito.
2. Verificar cantidades y subtotales.
3. Cambiar cantidad y ver recálculo.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
