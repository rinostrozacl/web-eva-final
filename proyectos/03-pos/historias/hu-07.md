# HU-07: No vender sin stock

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** sistema  
**Quiero** impedir vender productos sin stock  
**Para** no comprometer ventas imposibles

## Criterios de aceptación

- [ ] Dado producto con stock 0, cuando intento agregarlo al carrito o confirmar, entonces la operación falla con mensaje claro.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Poner producto en stock 0.
2. Intentar vender desde POS.
3. Verificar bloqueo y mensaje de error.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
