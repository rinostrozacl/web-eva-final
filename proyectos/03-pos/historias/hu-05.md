# HU-05: Confirmar venta con total

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** cajero  
**Quiero** confirmar la venta con total calculado  
**Para** cerrar la transacción

## Criterios de aceptación

- [ ] Dado carrito con ítems, cuando confirmo venta, entonces se registra con total = suma de líneas.
- [ ] Dado venta confirmada, entonces el carrito se vacía o muestra éxito.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Armar carrito con productos conocidos.
2. Confirmar venta.
3. Verificar total correcto y mensaje de éxito.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
