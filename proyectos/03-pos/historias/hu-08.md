# HU-08: Listado ventas del día

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** cajero  
**Quiero** listar las ventas del día con filtro por fecha  
**Para** revisar el movimiento

## Criterios de aceptación

- [ ] Dado ventas registradas, cuando filtro por fecha de hoy, entonces veo ventas de ese día con total.
- [ ] Dado otra fecha, cuando filtro, entonces el listado cambia.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Registrar venta de prueba.
2. Abrir listado de ventas.
3. Filtrar por fecha y verificar venta visible con monto.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
