# HU-08: Comprobante de pasaje

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** pasajero  
**Quiero** ver un comprobante con los datos del pasaje  
**Para** tener evidencia de la compra

## Criterios de aceptación

- [ ] Dado un pasaje comprado, cuando abro comprobante o detalle, entonces veo origen, destino, fecha, asiento y precio.
- [ ] Dado el comprobante, cuando lo visualizo, entonces los datos coinciden con la compra.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Abrir detalle o comprobante de un pasaje.
2. Verificar todos los campos requeridos.
3. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
