# HU-OPC-02: Comprobante en PDF

## Obligatoriedad

Opcional (bonus)

## Historia de usuario

**Como** pasajero  
**Quiero** descargar o imprimir el comprobante en PDF  
**Para** llevar el pasaje sin conexión

## Criterios de aceptación

- [ ] Dado pasaje comprado, cuando solicito PDF, entonces se genera o descarga archivo.
- [ ] Dado el PDF, cuando lo abro, entonces contiene datos del pasaje.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Comprar o usar pasaje existente.
2. Clic en descargar/imprimir PDF.
3. Abrir archivo y verificar contenido.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
5. Bonus: **+5 pts** si cumple; **0** si no implementado (sin descuento).
