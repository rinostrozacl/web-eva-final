# HU-02: Gestionar viajes desde la web

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** operador  
**Quiero** crear, editar y eliminar viajes desde la web  
**Para** actualizar horarios y tarifas

## Criterios de aceptación

- [ ] Dado operador autenticado, cuando creo viaje en web, entonces aparece en listado.
- [ ] Dado viaje existente, cuando edito precio u horario, entonces se guarda.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Login como operador.
2. Crear viaje; verificar listado.
3. Editar precio u horario; guardar y verificar.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
