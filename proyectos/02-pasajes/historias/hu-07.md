# HU-07: Mis pasajes

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** pasajero autenticado  
**Quiero** ver mis pasajes comprados  
**Para** consultar mis viajes

## Criterios de aceptación

- [ ] Dado pasajes comprados por el usuario, cuando abro «mis pasajes», entonces veo listado con origen, destino, fecha y asiento.
- [ ] Dado usuario sin pasajes, cuando abro la sección, entonces veo estado vacío claro.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Login como pasajero con pasajes.
2. Abrir «mis pasajes».
3. Verificar datos del viaje y asiento.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
