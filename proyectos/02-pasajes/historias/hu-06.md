# HU-06: Bloquear asiento ya vendido

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** sistema  
**Quiero** bloquear un asiento ya vendido  
**Para** no vender el mismo lugar dos veces

## Criterios de aceptación

- [ ] Dado asiento vendido, cuando otro usuario intenta comprarlo, entonces la operación falla.
- [ ] Dado el intento en API, entonces responde 409 o equivalente.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Comprar un asiento específico.
2. Con otra sesión, intentar comprar el mismo asiento.
3. Verificar bloqueo en UI y/o 409 en API.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
