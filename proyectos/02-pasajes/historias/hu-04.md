# HU-04: Ver asientos disponibles

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** pasajero  
**Quiero** ver los asientos disponibles de un viaje  
**Para** elegir dónde sentarme

## Criterios de aceptación

- [ ] Dado un viaje seleccionado, cuando abro selección de asientos, entonces veo mapa o lista con estado libre/ocupado.
- [ ] Dado asientos ocupados, cuando miro la UI, entonces no aparecen como seleccionables.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Seleccionar un viaje de la búsqueda.
2. Abrir pantalla de asientos.
3. Verificar que se distinguen disponibles vs ocupados.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
