# HU-05: Filtrar anuncios

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** comprador  
**Quiero** filtrar por marca y rango de precio  
**Para** acotar la búsqueda

## Criterios de aceptación

- [ ] Dado anuncios variados, cuando filtro por marca, entonces solo veo esa marca.
- [ ] Dado rango de precio, cuando aplico filtro, entonces solo veo anuncios dentro del rango.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Aplicar filtro marca.
2. Aplicar filtro precio min/max.
3. Verificar resultados coherentes.
4. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
