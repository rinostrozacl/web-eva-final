# HU-02: Publicar anuncio

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** vendedor autenticado  
**Quiero** publicar un anuncio desde la web  
**Para** ofrecer mi auto

## Criterios de aceptación

- [ ] Dado usuario autenticado, cuando completo formulario y guardo, entonces el anuncio queda publicado.
- [ ] Dado anuncio publicado, entonces aparece en listado público.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Login como vendedor.
2. Ir a publicar anuncio.
3. Completar marca, modelo, año, precio, km; guardar.
4. Ver confirmación y anuncio en listado.
5. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
