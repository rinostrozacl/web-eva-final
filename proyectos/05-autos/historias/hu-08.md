# HU-08: Solo el dueño edita

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** sistema  
**Quiero** que solo el dueño pueda editar o eliminar su anuncio  
**Para** proteger publicaciones ajenas

## Criterios de aceptación

- [ ] Dado anuncio de otro usuario, cuando intento editar o eliminar, entonces recibo 403 o UI bloqueada.
- [ ] Dado mi anuncio, cuando edito, entonces sí se permite.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Login usuario A; crear anuncio.
2. Login usuario B; intentar editar anuncio de A.
3. Verificar bloqueo.
4. Usuario A sí puede editar el suyo.
5. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
