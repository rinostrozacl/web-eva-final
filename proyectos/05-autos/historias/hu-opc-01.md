# HU-OPC-01: Fotos en anuncio

## Obligatoriedad

Opcional (bonus)

## Historia de usuario

**Como** vendedor  
**Quiero** agregar una o más fotos (URL) al anuncio  
**Para** mostrar el estado del vehículo

## Criterios de aceptación

- [ ] Dado anuncio, cuando agrego URL de foto, entonces se muestra en detalle y/o listado.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Editar o crear anuncio con URL de imagen.
2. Verificar que la imagen se renderiza.
3. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
4. Bonus: **+5 pts** si cumple; **0** si no implementado (sin descuento).
