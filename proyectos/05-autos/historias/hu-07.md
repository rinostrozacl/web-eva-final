# HU-07: Marcar como vendido

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** vendedor  
**Quiero** marcar un anuncio como vendido  
**Para** retirarlo del listado activo

## Criterios de aceptación

- [ ] Dado mi anuncio activo, cuando marco vendido, entonces deja de aparecer en listado activo.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Marcar anuncio como vendido.
2. Verificar que no aparece en listado público activo.
3. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
