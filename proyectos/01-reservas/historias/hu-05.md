# HU-05: Listar reservas por fecha

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** administrador  
**Quiero** ver las reservas filtradas por fecha  
**Para** organizar la agenda del día

## Criterios de aceptación

- [ ] Dado reservas existentes, cuando filtro por una fecha, entonces solo veo reservas de ese día.
- [ ] Dado el filtro, cuando cambio la fecha, entonces el listado se actualiza.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Login como administrador.
2. Ir a listado de reservas.
3. Seleccionar fecha con reservas conocidas.
4. Verificar que solo aparecen reservas de esa fecha.
5. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
