# HU-08: Vista agenda del día

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** administrador  
**Quiero** una vista de agenda del día con todas las citas  
**Para** visualizar la jornada de un vistazo

## Criterios de aceptación

- [ ] Dado reservas en un día, cuando abro la agenda de ese día, entonces veo todas las citas ordenadas por hora.
- [ ] Dado la vista agenda, cuando selecciono otro día, entonces se actualiza el contenido.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Login como administrador.
2. Abrir vista agenda o calendario del día.
3. Seleccionar fecha con reservas.
4. Verificar que muestra servicio, hora y cliente (o identificador) por cita.
5. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
