# HU-06: Cancelar reserva

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** cliente  
**Quiero** cancelar una reserva existente  
**Para** liberar el horario si no puedo asistir

## Criterios de aceptación

- [ ] Dado una reserva activa propia, cuando cancelo, entonces el estado cambia a cancelada.
- [ ] Dado la reserva cancelada, cuando otro cliente busca ese slot, entonces el horario vuelve a estar disponible.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Crear o usar reserva activa de prueba.
2. Cancelar desde la web.
3. Confirmar estado cancelado en UI.
4. Verificar que el slot queda disponible para nueva reserva.
5. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
