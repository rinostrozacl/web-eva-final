# HU-02: Gestionar servicios desde la web

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** administrador  
**Quiero** crear, editar y eliminar servicios desde la web  
**Para** mantener mi catálogo actualizado

## Criterios de aceptación

- [ ] Dado usuario autenticado como admin, cuando creo un servicio desde web, entonces aparece en el listado.
- [ ] Dado un servicio existente, cuando lo edito, entonces los cambios se guardan.
- [ ] Dado un servicio sin reservas activas (o según regla del grupo), cuando lo elimino, entonces desaparece del listado.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Login en producción.
2. Ir a gestión de servicios.
3. Crear servicio; verificar en listado.
4. Editar nombre o precio; guardar y verificar.
5. Eliminar o desactivar un servicio de prueba; verificar que ya no aparece (o marca inactivo).
6. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
