# HU-04: Reservar una cita

## Obligatoriedad

Obligatoria

## Historia de usuario

**Como** cliente  
**Quiero** reservar una cita eligiendo servicio, fecha y hora  
**Para** agendar mi atención

## Criterios de aceptación

- [ ] Dado servicios y horarios disponibles, cuando elijo servicio, fecha y hora y confirmo, entonces la reserva queda creada.
- [ ] Dado la reserva creada, cuando consulto mis reservas o agenda, entonces aparece con los datos correctos.

## Requiere interfaz web

Sí

## Procedimiento de verificación (corrector)

1. Login o flujo de cliente.
2. Elegir servicio, fecha y hora disponible.
3. Confirmar reserva.
4. Ver mensaje de éxito y reserva visible en listado o agenda.
5. Marcar **cumple (2)** si todos los pasos se completan; **parcial (1)** si falta web donde se exige; **no cumple (0)** en caso contrario.
